## 🧑‍💻 How to Use

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/sg7373/Portfolio-7373
cd your-repo-name
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Run the App

```bash
npm start
```

### 4️⃣ Build for Production

```bash
npm run build
```
